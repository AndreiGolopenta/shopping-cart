# Shooping Cart

## Description

Simple e-commerce cart application build with:
* Angular
* NGRX

Actions that can be made:
* add/remove/update the cart;
* filter the products;
* sorting the products;
* searching products;

## Requirements

* JSON Server

### JSON Server

Install JSON Server : 
```cli
npm install -g json-server
```

## Project Dependencies

`npm install`

## Running the project

```cli
json-server --watch db.json
```
```cli
npm run start
```
